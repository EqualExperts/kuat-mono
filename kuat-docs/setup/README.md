## Quick Start

1. **Read the rules:** Start with `rules/` directory for design language
2. **Choose your framework:** Load relevant `examples/` if you need code
3. **Integrate:** Follow [integration.md](./integration.md) for your environment
4. **Verify:** Test with [verification.md](./verification.md)

---

## What to Load

**For pure design rules (no code):**
- Load `rules/` directory only
- ~1500 lines total across all rules files

**For framework-specific development:**
- Load `rules/` + your framework from `examples/`
- ~2000 lines total

**For full context:**
- Load everything in `kuat-docs/`
- ~4000 lines total
