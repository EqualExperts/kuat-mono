# Content Rules

Content writing guidelines for the Kuat Design System. These rules ensure consistent voice, tone, and messaging across all Equal Experts communications.

---

| File | Description | Audience |
|------|-------------|----------|
| [foundations.md](./foundations.md) | Universal voice and tone principles | Everyone |
| [marketing-sales.md](./marketing-sales.md) | Marketing, sales, and external content | Marketing, Sales, PR |
| [product-ux.md](./product-ux.md) | Product interface and UX content | UX Writers, Designers, Developers |

---

## Reading Order

1. **Start with [foundations.md](./foundations.md)** - Core principles apply to all content
2. **Then read the specialized guide** for your content type

---

## Key Principles

1. **User-first** - Write for the reader, not the organization
2. **Clarity** - Clear language over clever language
3. **Conciseness** - Respect the reader's time
4. **Consistency** - Same voice across all touchpoints
5. **Testing** - Validate content with real users

---

## Quick Reference

**Voice**: Confident, knowledgeable, approachable, direct

**Tone adaptation**:
- Marketing: Confident to conversational
- Product: Helpful, concise, action-oriented
- Internal: Friendly, authentic, generous
