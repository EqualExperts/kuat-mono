## Key Principles

1. **Consistency** - Use design tokens, not hardcoded values
2. **Accessibility** - WCAG AA compliance minimum
3. **Simplicity** - Minimal borders, clear hierarchy through spacing
4. **Flexibility** - Light and dark mode support
